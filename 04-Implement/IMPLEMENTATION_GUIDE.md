# Implementation Guide

All AI implementations and human developer tasks must be documented and stored in the subdirectories here:
- **AI_TASKS**: Document individual AI tasks here.
- **IMPLEMENTATION_REPORTS**: Store final reports for features developed.
- **DATABASE_MIGRATION_REPORTS**: Keep logs of all database structural changes.
- **REFACTOR_REPORTS**: Document structural and code smell refactoring.
- **CHANGE_LOGS**: Daily or feature-based changelogs.
